# Blog

Read fake posts and comments, search for all users and check your informations

## About

List of all features implemented in this application: 

- Lists all posts
- Lists all comments
- Search for any user
- Lists informations about user

## How to run

1. Clone this repository
2. Access the directory
```bash
cd blog/
```
3. Install dependencies
```bash
npm i
```
4. Start application with 
```code
  npm run dev
```
5. Access `http://localhost:5173` and enjoy
